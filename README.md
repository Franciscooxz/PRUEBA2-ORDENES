# API de Órdenes · Go + GraphQL + PostgreSQL + JWT

API de gestión de órdenes de compra. Maneja usuarios, productos y órdenes con sus
ítems, con autenticación JWT stateless y creación de órdenes transaccional.

> Prueba técnica 2 — Wind Consulting.

## Stack

- **Go 1.26**
- **gqlgen** (GraphQL)
- **PostgreSQL** vía **pgx**
- **JWT** para autenticación
- **DataLoader** para resolver el problema N+1
